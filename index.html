Profesyonel Web Portu

Terminal korku deneyimi artık mobil uyumlu.

Bu sürüm; sahne akışı, CRT glitch mantığı, geçiş statikleri, ses motoru davranışı ve CTRL+C alternatif senaryosunu modern web mimarisine taşır.

Uyumluluk

Android, iPhone, iPad, PC, GitHub Pages ve Termux WebView içinde çalışacak şekilde tasarlandı.

Etkileşim

Masaüstünde Ctrl+C, mobildeyse kırmızı tetik butonu ile alternatif senaryo başlatılır.

Motor Karşılığı

SceneManager, GlitchEngine, AudioManager ve görsel elementler DOM + Canvas + Web Audio davranışına çevrildi.
